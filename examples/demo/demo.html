GeoFire

powered by Firebase

What is GeoFire?

geoFire is a helper library for location-based operations in Firebase. It provides functions to store data for location querying in Firebase, and perform location queries such as location updates and localized searches.

About the demo

The example is using the onPointsNearLoc function in geoFire to update an area of the map in realtime.

API Functions

insertByLoc()
Inserts data solely by location, specified by a [latitude, longitude] array.
insertByLocWithId()
Inserts data by location, specified by a [latitude, longitude] array, and a client-provided identifer.
removeById()
Removes the data point with the specified Id; the data point must have been inserted using insertByLocWithId.
getLocById()
Gets the location of the data point with the specified Id; the data point must have been inserted using insertByLocWithId.
updateLocForId()
Updates the location of the data point with the specified Id; the data point must have been inserted using insertByLocWithId.
getPointsNearLoc()
Finds all data points that are within the specified radius from the source point, specified by a [latitude, longitude] array, at the time the function is called.
onPointsNearLoc()
As above but continues to return matching data points till offPointsNearLoc is called.
offPointsNearLoc()
getPointsNearId()
Finds all data points that are within the specified radius from the source point, specified by Id, at the time the function is called.
onPointsNearId()
As above but continues to return matching data points till offPointsNearId is called.
offPointsNearId()
